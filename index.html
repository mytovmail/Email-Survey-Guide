ניהול סקרים בעזרת בינה מלאכותית

המערכת תוכננה כך שתהיה חכמה וגמישה. אתם לא צריכים לדעת לכתוב אפילו שורת קוד אחת! כדי לבצע שינויים, פשוט נעזרים בבינה מלאכותית ועוקבים אחר השלבים הבאים:

במקום להסתבך עם עריכת קוד ה-HTML בעצמכם, פעלו לפי 3 השלבים הפשוטים הבאים:

  1. העתקת הקוד הקיים: גשו לגיליון "הגדרות", לחצו פעמיים על תא B8 (התא האפור עם הקוד) והעתיקו את כל הטקסט שבתוכו.
  2. בקשו מהבינה המלאכותית: פתחו את כלי ה-AI המועדף עליכם (ChatGPT, קלוד וכו'), והדביקו לו את אחת מ"תבניות הבקשה" הבאות:
    💬 תבנית להוספת כפתור (למשל: "גרוע")

    הנה קוד HTML של סקר במייל. [הדבק כאן את הקוד שהעתקת מתא B8] אנא הוסף לכל השאלות כפתור הצבעה רביעי. הטקסט על הכפתור יהיה 'גרוע', והערך של ההצבעה שלו בקישור צריך להיות vote=4. החזר לי את הקוד המלא והמתוקן בבקשה.

    💬 תבנית להוספת שאלת סקר חדשה

    הנה קוד HTML של סקר במייל. [הדבק כאן את הקוד שהעתקת מתא B8] אנא הוסף שאלה חדשה לתחתית הסקר. השאלה תהיה: 'איך תדרג את מהירות המענה שלנו?'. מזהה הסקר שלה בקישור (poll_id) יהיה 'שאלה_11'. הוסף לה 3 כפתורים בדיוק כמו בשאר השאלות בסקר (מצוין=1, בינוני=2, טעון שיפור=3). החזר לי את הקוד המלא והמתוקן.

  3. הדבקה חזרה בגיליון: העתיקו את הקוד החדש שהבינה המלאכותית יצרה, והדביקו אותו חזרה בתוך תא B8 (במקום התוכן הישן).

לאחר שהדבקתם את הקוד החדש, צריך לעדכן את המערכת כדי שתדע לקרוא את השינויים שעשה ה-AI:

אם הוספתם שאלות חדשות

פעולת קסם אוטומטית:

  • • גשו לתפריט העליון בגיליון ולחצו על: "מערכת סקרים 📊" > "2. הוסף מילון לסקר חדש ➕".
  • • המערכת תסרוק אוטומטית את הקוד, תזהה את השאלות החדשות, ותייצר עבורן שורות מסודרות במילון בתחתית הגיליון! (כעת רק ודאו שהתרגום בעמודה C תקין).

אם הוספתם כפתור חדש

למשל כפתור מספר 4 ("גרוע"):

  • • רדו לחלק של "מילון כפתורים" (הטבלה למטה בעלת 3 העמודות) והוסיפו שורה חדשה וריקה בתחתית.
  • • עמודה A: הקלידו כללי
  • • עמודה B: המספר החדש (למשל 4)
  • • עמודה C: הטקסט שיוצג (למשל גרוע)

כדי לראות את התוצאות בצורה ויזואלית ומרשימה, אין צורך לחשב שום דבר ידנית:

1. ודאו שיש לכם כבר מספר הצבעות בגיליון "הצבעות".

2. גשו לתפריט העליון: "מערכת סקרים 📊" > "4. הפק דשבורד גרפים 📈".

3. המערכת תייצר עבורכם באופן אוטומטי לשונית חדשה בשם "דשבורד_גרפים".

4. בלשונית זו תמצאו טבלאות סיכום וגרף עוגה תלת-ממדי לכל שאלה וסקר בנפרד, מוכנים להצגה או להדפסה!

רוצים לשלוח את הסקר לאלפי מנויים דרך מערכת הדיוור המקצועית שלכם (כמו פולסים, סמוב או רב-מסר)? גם כאן, אין צורך לגעת בקוד - תנו לבינה המלאכותית להכין אותו עבורכם!

  1. הכנת הנתונים: העתיקו את קוד ה-HTML של הסקר מתא B8. בנוסף, העתיקו את "כתובת השרת" שלכם (הקישור) מתוך תא B2 בגיליון ההגדרות.
  2. בקשו מהבינה המלאכותית להתאים את הקוד:
    💬 תבנית התאמה למערכת דיוור

    הנה קוד HTML של סקר במייל: [הדבק כאן את הקוד מתא B8] אני רוצה לשלוח את הסקר דרך מערכת הדיוור שלי ([ציין את שם המערכת, למשל: פולסים / רב-מסר / סמוב]). אנא בצע עבורי 2 שינויים בקוד: 1. החלף את כל המילים {{SERVER_URL}} בכתובת השרת שלי: [הדבק כאן את הקישור מתא B2]. 2. החלף את כל המילים {{EMAIL}} בתגית המייל הדינמית של מערכת הדיוור שלי (למשל ##Email## בפולסים, או [email] בסמוב). החזר לי את הקוד המוכן להעתקה.

  3. הדבקה במערכת הדיוור: העתיקו את הקוד המוכן שהבינה המלאכותית החזירה לכם. כעת, צרו קמפיין חדש במערכת הדיוור שלכם, והדביקו את הקוד לתוך רכיב של "בלוק קוד" (HTML Block). זהו, הכל מוכן לשליחה וההצבעות יזרמו אוטומטית לגיליון!
